# 🚀 Smart Resume Analyzer

An end-to-end Machine Learning web application that analyzes resume text and predicts the most suitable job role using Natural Language Processing (NLP) techniques.

This project demonstrates a complete ML workflow including data preprocessing, feature extraction, model training, evaluation, backend integration, and cloud deployment using a Flask-based web application.

---

## 🔍 Features

- 📄 Resume text analysis and job role prediction  
- 🤖 Machine Learning-based classification system  
- ⚡ Fast and interactive web interface  
- 🌐 Deployed on cloud (Render) for real-time access  
- 🔄 End-to-end ML pipeline from training to deployment  

---

## 🧠 Machine Learning Workflow

- Data collection and labeling of resumes with job roles  
- Text preprocessing and cleaning  
- Feature extraction using **TF-IDF Vectorization**  
- Model training using **Logistic Regression**  
- Model evaluation and optimization  
- Model serialization using **pickle** for reuse  

---

## ⚙️ Tech Stack

- **Python**
- **Flask (Backend Framework)**
- **Scikit-learn (Machine Learning)**
- **TF-IDF (NLP Feature Extraction)**
- **HTML/CSS (Frontend)**
- **Render (Cloud Deployment)**

---

## 🔄 How It Works

1. User enters resume text into the web interface  
2. Flask backend receives the input  
3. Text is transformed using TF-IDF vectorizer  
4. Trained ML model predicts the job role  
5. Result is displayed dynamically on the UI
